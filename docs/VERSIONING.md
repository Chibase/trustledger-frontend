## Version 002 (active build — TEDS core pain)

**Goal:** Close the gap vs market tools on the modules buyers actually need.

| Domain | Outcome |
|--------|---------|
| **Geo** | SA hierarchy (province → municipality → ward+) + link records to place; ingest your location + socio-economic layers |
| **Stakeholders** | Registry (people, orgs, influence, interests) |
| **Engagements** | Meetings, consultations, attendance, minutes/actions |
| **Commitments** | Promise → owner → deadline → evidence → closed |
| **Stronger grievance** | Fuller lifecycle on Frappe (categorise → escalate → investigate → resolve → verify) |
| **Reports** | Operational + executive packs from live/geo/socio-economic data |
| **Intelligence / ESG** | Indicators, trust/ESG depth, stronger AI, executive packs |

**Public line:**  
*Version 002 — Stakeholder Intelligence core: registry, engagements, commitments, geo & ESG depth. In active development; Version 001 remains the live desk.*
